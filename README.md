# Cognizant-DN5 — Final Project

This repository contains exercises and a final project for Cognizant Upskilling.

How to view the final project:

- Open `Final-Project/index.html` in a browser (double-click or serve with a local HTTP server).

If the project uses Node/npm (in `Final-Project`):

```bash
cd Final-Project
npm install
npm start
```

To contribute:

```bash
git add .
git commit -m "describe change"
git push origin main
```
